## What This Session Covers

In this introductory video, Ali walks you through what the course is about, who it's for, and what you'll be able to build by the end. No Dynamo or scripting experience is required — just a working install of Revit.

**Topics covered:**
- What is Dynamo and why it matters for BIM professionals
- Who this course is for (Architecture, MEP, Structural, BIM Management)
- How the course is structured (3 levels, 30 sessions, 20 min each)
- How to use this GitHub repository alongside the videos
- Prerequisites and software setup

## Getting Started

Before Session 01, make sure you have:

- [ ] Autodesk Revit 2020 or later installed
- [ ] Dynamo is available inside Revit (Manage tab → Dynamo)
- [ ] This repository cloned or downloaded to your machine

```
git clone https://github.com/AliR1371/Dynamit.git
```

Or download as a ZIP from the green **Code** button above.
