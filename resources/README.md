## Dynamo Version Compatibility

| Revit Version | Dynamo Version | Compatibility |
|---|---|---|
| Revit 2024 | Dynamo 2.18 | ✅ Fully compatible |
| Revit 2023 | Dynamo 2.16 | ✅ Fully compatible |
| Revit 2022 | Dynamo 2.13 | ✅ Scripts built on this version |
| Revit 2021 | Dynamo 2.10 | ✅ Compatible (minor UI differences) |
| Revit 2020 | Dynamo 2.6 | ⚠️ Mostly compatible — some nodes differ |
| Revit 2019 and earlier | Dynamo 1.x | ❌ Not supported |

All scripts in this course are built and tested on **Dynamo 2.13 (Revit 2022)**. They are expected to work on any Dynamo 2.x version.
