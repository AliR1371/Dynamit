## About This Course

This repository is the companion resource for the **Dynamo for Revit** YouTube series — a structured, project-focused course that takes you from opening Dynamo for the first time to building professional-grade automation scripts used in real BIM workflows.

Every session is **20 minutes**, focused on one clear skill, and includes:
- Explained learning objectives
- Key nodes and concepts
- A practical, copy-ready Dynamo script

The course covers **four disciplines** throughout:

| Discipline | Colour Tag |

| Architecture | 🟦 |
| MEP | 🟩 |
| Structural | 🟥 |
| BIM Management | 🟨 |

## How to Use This Repository

Each session folder contains:

```
sessions/
└── 01-introduction-to-dynamo-for-revit/
    ├── README.md        ← Session notes, objectives, key nodes
    └── scripts/
        └── 01_Hello_World.dyn   ← Dynamo script (.dyn)
```

**To use a script:**
1. Open Revit with a sample project
2. Launch Dynamo (Manage tab → Dynamo)
3. Open the `.dyn` file from the session's `scripts/` folder
4. Follow along with the video

**Dynamo version compatibility:** All scripts are built and tested on **Dynamo 2.13+** (Revit 2022 and above). Most scripts work on Dynamo 2.x from Revit 2020+.
